Este proyecto fue hecho para un Curso de JavaScript en Tecno3F, parte de la municipalidad de Tres de Febrero.

Este código trata de una página de venta de películas usando la API de TheMovieDB, aplicando cosas vistas en el curso, por ejemplo promesas, funciones, bucles repetitivos, etc.
cuenta con 4 secciones (Inicio, Carrito, Pago y Contacto) encontramos lo siguiente:

Inicio: listado de todas las películas que tiene la API con un botón de agregar al carrito y el precio
Carrito: Listado de las películas agregadas al carrito
Pago: Formulario de pago en el cual solicita nombre del titular, numero de tarjeta, Código de verificación y fecha de expiración
Contacto: Formulario de contacto, dejando al usuario agregar su nombre, su mail, el asunto y un mensaje
